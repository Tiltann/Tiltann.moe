# tiltann

my personal site. one-column dev page.

## run it

```sh
npm install
npm run dev
```

open http://localhost:5173

## stack

- vue 3 with `<script setup>`
- vite
- tailwind
- JetBrains Mono (primary) + Minecraft/Monocraft (display)

## deploy

`npm run build` → `dist/`.

## How this is hosted

This is hosted on Cloudflare Pages under tiltann.de
